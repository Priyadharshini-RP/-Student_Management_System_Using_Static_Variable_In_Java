# -Student_Management_System_Using_Static_Variable_In_Java

🚀 **Java Student Management Program**

I created a Java program that stores and displays student details using Object-Oriented Programming concepts.

🔹 Features:
• Uses a **Student class** with attributes like name, ID, department, and age
• Implements a **constructor** to initialize student details
• Uses a **static variable** to store the college name for all students
• Accepts **multiple student inputs using Scanner**
• Stores objects in an **array** and displays the information using a method

💡 Concepts used: Java Classes, Objects, Constructors, Static Variables, Arrays, and User Input.
